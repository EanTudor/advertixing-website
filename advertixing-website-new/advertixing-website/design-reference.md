# Current Advertixing.com Design Reference

## Visual Design System
- **Background:** Pure black (#000000 or very close)
- **Text:** White (#FFFFFF) for headlines, lighter gray for body
- **Accent color:** Cyan/electric blue (#00BFFF or similar bright cyan) — used for the "X" in logo, section labels, stats numbers, CTA buttons
- **Secondary accent:** Bright cyan for buttons ("Book a Call" button is cyan/teal)
- **Cards:** Dark gray (#1a1a1a or similar) with subtle border, rounded corners
- **Font:** Bold sans-serif for headlines (looks like Inter or similar), regular weight for body
- **Logo:** "AdvertiXing" with the "X" in cyan/blue

## Layout Structure
- **Navbar:** Fixed top, transparent/dark, logo left, nav links center, CTA button right (cyan)
- **Hero:** Full-width, centered layout, pill badge above headline, large bold headline, subtitle, two CTA buttons, stats row below

## Sections (in order)
1. **Hero** — Centered, pill badge "PAID MEDIA AGENCY — X ADS SPECIALISTS", large headline, subtitle, two buttons (filled cyan + outlined), stats row (3 stats in a dark card)
2. **Services** — "WHAT WE DO" label, centered headline, 4 service cards in a row (dark cards with icon, title, description)
3. **Who We Serve** — "WHO WE SERVE" label, centered headline, 4 client type cards with numbered badges (01-04), bullet points with → arrows
4. **Process** — "HOW IT WORKS" label, numbered steps (1-4) in a vertical list with large number badge, title, description
5. **Results** — "RESULTS" label, centered headline, 4 stat cards (large colored number, description, category tag)
6. **Testimonials** — Carousel with dots, dark card with testimonial content
7. **Final CTA** — Centered, large headline, description, CTA button
8. **Footer** — Centered logo, tagline, nav links, copyright + email

## Key Design Details
- Section labels are ALL CAPS, small, colored (cyan)
- Headlines are large, bold, centered
- Cards have dark background, rounded corners (~8-12px), subtle border
- Numbers in results are large and cyan/blue colored
- Process steps have large numbered circles on the left
- "Book a Call" CTA button is consistently cyan/teal, rounded, prominent
- Overall: Dark, modern, centered layout — not asymmetric
- Spacing is generous between sections
